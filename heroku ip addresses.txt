How to config custom domain for heroku web app
----------------------------------------------

1. create rails projects
2. push git heroku master
3. add custom domain addons in web-base interface or command lines
	heroku addons:add custom_domains
	heroku domains:add <domain.com>
	heroku domains:add <www.domain.com>

4. go to domain setting in domain hosting providers (godday, enom, name, namecheap, ...)

	add the following IPs to A Record:

	@  ->  75.101.163.44
	@  ->  75.101.145.87
	@  ->  174.129.212.2
	
	and add the CName with
	
	www -> <appname>.(heroku/heorkuapp).com
	
after pointing the address to the heroku IPs, we can check if the addresses taken in to effect by:

	host <domain.com>
